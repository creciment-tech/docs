## Overview

Fetch detailed information about a specific transaction using either its `id` (UUID) or `reference`.

<Note>
  **Required permission:** `payment:read`
</Note>

## Endpoint

```
GET /api/payments/get-transaction
```

## Request

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your Chidori API key
</ParamField>

### Query parameters

Provide either `id` OR `reference`:

<ParamField query="id" type="string">
  The transaction ID (UUID)
</ParamField>

<ParamField query="reference" type="string">
  The transaction reference string
</ParamField>

## Response

### Success response

<ResponseField name="status" type="boolean">
  Always `true` for successful requests
</ResponseField>

<ResponseField name="data" type="object">
  Complete transaction object with all details
</ResponseField>

## Examples

<RequestExample>
```bash cURL (by ID)
curl "https://api.chidori.africa/enterprise-service/api/payments/get-transaction?id=txn-123-456-789" \
  -H "X-API-Key: sk_live_xxx.yyy"
```

```bash cURL (by reference)
curl "https://api.chidori.africa/enterprise-service/api/payments/get-transaction?reference=order_12345_payment" \
  -H "X-API-Key: sk_live_xxx.yyy"
```

```javascript JavaScript
// By reference
const response = await fetch(
  'https://api.chidori.africa/enterprise-service/api/payments/get-transaction?reference=order_12345_payment',
  {
    headers: {
      'X-API-Key': 'sk_live_xxx.yyy'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

# By ID
response = requests.get(
    'https://api.chidori.africa/enterprise-service/api/payments/get-transaction',
    params={'id': 'txn-123-456-789'},
    headers={'X-API-Key': 'sk_live_xxx.yyy'}
)

data = response.json()
```
</RequestExample>

## Use cases

<AccordionGroup>
  <Accordion title="Verify payment status">
    After processing a payment, verify it was successful:
    
    ```javascript
    async function verifyPayment(reference) {
      const response = await fetch(
        `https://api.chidori.africa/enterprise-service/api/payments/get-transaction?reference=${reference}`,
        { headers: { 'X-API-Key': apiKey } }
      );
      
      const { status, data } = await response.json();
      
      if (status && data) {
        return {
          verified: true,
          amount: data.amount,
          timestamp: data.createdAt
        };
      }
      
      return { verified: false };
    }
    ```
  </Accordion>
  
  <Accordion title="Reconciliation">
    Match transactions with your internal records:
    
    ```javascript
    async function reconcileOrder(orderId) {
      const reference = `order_${orderId}_payment`;
      
      const response = await fetch(
        `https://api.chidori.africa/enterprise-service/api/payments/get-transaction?reference=${reference}`,
        { headers: { 'X-API-Key': apiKey } }
      );
      
      const { data } = await response.json();
      
      if (data) {
        await updateOrderRecord(orderId, {
          paymentVerified: true,
          transactionId: data.id,
          paidAmount: data.amount,
          paidAt: data.createdAt
        });
      }
    }
    ```
  </Accordion>
</AccordionGroup>
