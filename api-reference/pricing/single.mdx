## Overview

Calculate the delivery price estimate for a single pickup-to-destination route. This endpoint returns pricing information and a `tempDeliveryId` that you use to create the actual delivery.

<Note>
  **Required permission:** `pricing:read`
</Note>

## Endpoint

```
POST /api/pricing/single
```

## Request

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your Chidori API key
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body parameters

<ParamField body="pickup" type="object" required>
  Pickup location coordinates
  <Expandable title="properties">
    <ParamField body="lat" type="number" required>
      Latitude of the pickup location (e.g., `6.5244`)
    </ParamField>
    <ParamField body="lon" type="number" required>
      Longitude of the pickup location (e.g., `3.3792`)
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="destination" type="object" required>
  Destination location coordinates
  <Expandable title="properties">
    <ParamField body="lat" type="number" required>
      Latitude of the destination (e.g., `6.4654`)
    </ParamField>
    <ParamField body="lon" type="number" required>
      Longitude of the destination (e.g., `3.4064`)
    </ParamField>
  </Expandable>
</ParamField>

## Response

### Success response

<ResponseField name="status" type="boolean">
  Always `true` for successful requests
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="distanceKm" type="number">
      Distance in kilometers between pickup and destination
    </ResponseField>
    <ResponseField name="durationSec" type="number">
      Estimated duration in seconds
    </ResponseField>
    <ResponseField name="geometry" type="string">
      Encoded polyline of the route (can be null)
    </ResponseField>
    <ResponseField name="tempDeliveryId" type="string">
      Temporary delivery ID to use when creating the delivery
    </ResponseField>
    <ResponseField name="pricing" type="object">
      <Expandable title="properties">
        <ResponseField name="distance_km" type="number">
          Distance in kilometers
        </ResponseField>
        <ResponseField name="total_cost" type="number">
          Total delivery cost in Naira
        </ResponseField>
        <ResponseField name="promo_applied" type="boolean">
          Whether a promotional discount was applied
        </ResponseField>
        <ResponseField name="promo_percent" type="number">
          Percentage discount applied (0 if none)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<RequestExample>
```bash cURL
curl -X POST https://api.chidori.africa/api/pricing/single \
  -H "X-API-Key: sk_live_xxx.yyy" \
  -H "Content-Type: application/json" \
  -d '{
    "pickup": {
      "lat": 6.5244,
      "lon": 3.3792
    },
    "destination": {
      "lat": 6.4654,
      "lon": 3.4064
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.chidori.africa/api/pricing/single', {
  method: 'POST',
  headers: {
    'X-API-Key': 'sk_live_xxx.yyy',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    pickup: { lat: 6.5244, lon: 3.3792 },
    destination: { lat: 6.4654, lon: 3.4064 }
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.chidori.africa/api/pricing/single',
    headers={
        'X-API-Key': 'sk_live_xxx.yyy',
        'Content-Type': 'application/json'
    },
    json={
        'pickup': {'lat': 6.5244, 'lon': 3.3792},
        'destination': {'lat': 6.4654, 'lon': 3.4064}
    }
)

data = response.json()
```
</RequestExample>

## Error codes

| Status | Message | Description |
|--------|---------|-------------|
| `400` | Location out of bounds | Pickup or destination is outside the supported service area |
| `401` | Unauthorized | Invalid or missing API key |
| `403` | Forbidden | API key lacks `pricing:read` permission |
